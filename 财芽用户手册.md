## App功能概览

### 五大核心功能

#### 1. 行情
查看基金涨跌排行、指数行情、今日收益

#### 2. 持仓

## 快速入门

### 第一步：添加自选基金

1. 打开 App，点击底部 **"行情"** 标签
2. 在涨跌幅榜中，点击任意基金名称
3. 进入基金详情页面，点击右上角 **⭐ 星标图标**
4. 该基金添加到自选列表

### 第二步：买入基金（记录持仓）

**注意**：本 App 不能真实购买基金，只是记录你的交易

1. 在基金详情页面，点击底部 **"买入"** 按钮
2. 填写买入信息：
   - **购买日期**：你实际买入的日期（如 2024-09-24）
   - **购买金额**：你花了多少钱（如 5000）
3. 点击 **"确认买入"**
4. 系统自动计算份额，添加到持仓

### 第三步：查看持仓

1. 点击底部 **"持仓"** 标签
2. 查看你的所有基金持仓
3. 顶部显示：
   - **总盈亏**：赚了或亏了多少钱
   - **总成本**：买入花的总金额
   - **总市值**：现在值多少钱

### 可视化功能

#### 功能1：估值温度计

**功能介绍**：判断当前指数是高估还是低估

温度计从左到右分为5个区域：
```
[  绿色  |  浅绿  |  黄色  |  橙色  |  红色  ]
  低估     偏低     正常     偏高    高估
  0-20%  20-40%  40-60%  60-80%  80-100%
```

**PE百分位和PB百分位**：
- 这是衡量指数估值的两个指标
- 百分位越低 = 越便宜 = 越适合买入
- 百分位越高 = 越贵 = 可能需要谨慎

**实际应用**：
- 温度在 **绿色区域**（0-40%）→ 可以考虑买入
- 温度在 **黄色区域**（40-60%）→ 正常估值
- 温度在 **红色区域**（80-100%）→ 可能偏贵，谨慎买入

## 常见操作指南

### 如何买入基金？

**步骤详解**：

1. **找到想买的基金**
   - 搜索基金代码或名称或在行情、自选页面点开基金卡片

2. **点击"买入"按钮**
   - 位置：页面底部

3. **填写买入信息**
   ```
   购买日期: 2025-10-22  (选择你实际买入的日期)
   购买金额: 5000        (输入金额，单位：元)
   ```

4. **确认买入**
   - 点击"确认买入"按钮
   - 系统自动计算份额
   - 添加到持仓列表

---

### 如何卖出基金？

**步骤详解**：

1. **进入基金详情页**
   - 从持仓列表点击要卖出的基金

2. **点击"卖出"按钮**
   - 位置：页面底部

3. **填写卖出信息**
   ```
   卖出份额: 1000  (输入要卖多少份)
   
   快捷按钮:
   [25%] [50%] [75%] [全部]
   ```

4. **确认卖出**
   - 点击"确认卖出"按钮
   - 持仓自动减少

**重要说明**：
- 卖出采用 **先进先出** 原则（最早买的先卖）
- 如果卖出全部份额，该基金从持仓列表消失
- 卖出记录会保存在"记录"页面

---

## 常见问题解答

### Q1: 今日收益是如何计算的？

**答**：基于实时估值计算

**计算公式**：
```
今日收益 = Σ (持仓份额 × 今日涨跌额)
```

**注意**：
- 今日收益是估算值，不是最终收益
- 只统计持仓基金，不包括已卖出的
- 待确认基金不计入今日收益

---

### Q2: 为什么有些基金搜索不到？

**答**：数据来源限制

- App数据来自东方财富网
- 仅包含公开发行的基金
- 不包含私募基金、理财产品

**建议**：
- 使用6位基金代码搜索（如 000001）
- 确认基金代码输入正确

---

### Q3: 基金详情中的信息来自哪里？

**答**：所有数据来自公开渠道

- **实时估值**：天天基金网
- **基金信息**：东方财富网
- **净值数据**：基金公司官网数据
- **估值温度**：指数公开数据

**数据刷新频率**：
- 实时估值：交易时间内实时更新
- 基金详情：60分钟缓存
- 历史净值：每日更新

---

### Q4: 数据会丢失吗？

**答**：数据保存在本地

- 所有数据存储在手机本地数据库
- 卸载App会清空数据
- 不支持云同步
